# Brand Guidelines Examples

Reference examples for different brand personalities.

## Example 1: Premium/Luxury Brand

**Brand**: Luxe Timepieces (fictional watch brand)

### Palette
- **Primary**: `#1a1a1a` (near-black)
- **Secondary**: `#c9a961` (muted gold)
- **Accent**: `#8b7355` (bronze)
- **Background**: `#faf9f7` (warm white)
- **Text**: `#2d2d2d`

### Typography
- **Headings**: Cormorant Garamond (elegant serif)
- **Body**: Lato (clean, readable)

### Characteristics
- High whitespace
- Subtle animations (fade, slow reveals)
- Minimal UI chrome
- Photography: high contrast, dramatic lighting

## Example 2: Playful/Consumer Brand

**Brand**: BounceBox (fictional kids' activity subscription)

### Palette
- **Primary**: `#ff6b6b` (coral red)
- **Secondary**: `#4ecdc4` (teal)
- **Accent**: `#ffe66d` (sunny yellow)
- **Background**: `#f7f7f7`
- **Text**: `#2d3436`

### Typography
- **Headings**: Fredoka One (rounded, friendly)
- **Body**: Nunito (soft, approachable)

### Characteristics
- Rounded corners (12-16px radius)
- Bouncy animations (spring easing)
- Illustrated icons
- Photography: bright, natural lighting, candid moments

## Example 3: Technical/B2B Brand

**Brand**: DataForge (fictional analytics platform)

### Palette
- **Primary**: `#0066cc` (confident blue)
- **Secondary**: `#00a67e` (success green)
- **Accent**: `#6366f1` (indigo)
- **Background**: `#f8fafc`
- **Text**: `#1e293b`

### Typography
- **Headings**: IBM Plex Sans (technical precision)
- **Body**: Inter (highly readable)
- **Mono**: JetBrains Mono (code blocks)

### Characteristics
- Sharp corners or minimal radius (2-4px)
- Subtle, functional animations
- Data visualization as decoration
- Clean iconography (2px stroke)

## Color Accessibility Quick Reference

### Contrast Ratios (WCAG 2.1)

| Level | Normal Text | Large Text | UI Components |
|-------|-------------|------------|---------------|
| AA | 4.5:1 | 3:1 | 3:1 |
| AAA | 7:1 | 4.5:1 | N/A |

### Safe Pairings

| Background | Foreground | Ratio | Level |
|------------|------------|-------|-------|
| `#ffffff` | `#1a1a1a` | 17.4:1 | AAA |
| `#f5f5f5` | `#333333` | 11.5:1 | AAA |
| `#0066cc` | `#ffffff` | 4.5:1 | AA |
| `#ff6b6b` | `#000000` | 5.2:1 | AA |

### Common Failures to Avoid

- Light gray text on white (`#999` on `#fff` = 2.8:1 ❌)
- Yellow on white (`#ffc107` on `#fff` = 1.5:1 ❌)
- Placeholder text too light
- Disabled states with insufficient contrast
